FKIK Editor

Project Info

C++ OpenGL Unity

Overview

This project is an interactive FKIK editor and Unity FKIK plugin developed using the Animation Toolkit.

Features

Forward Kinematics

• Load and play BVH files.
• Camera control: rotate, zoom, pan.
• Reset button loads default Beta.bvh motion.
• Time Scale slider controls animation speed.

Inverse Kinematics

• Select IK solver: Limb-based IK, CCD IK, Pseudo Inverse IK.
• Move targets: input fields or drag green points.

Unity Plugin

• Compile code as dynamic library "FKIKPlugin.dll" and load in Unity as native plugin.
• Camera control: move, rotate, zoom, pan.

Implementation

• Implemented FKIK editor and Unity plugin using Animation Toolkit.
• Developed forward and inverse kinematics functionality.
