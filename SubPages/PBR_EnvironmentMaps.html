Physically-Based Rendering (PBR) Shader

Project Info

C++, OpenGL, GLSL

Overview

This project implements a Physically-Based Rendering (PBR) shader using the Cook-Torrance BRDF and Lambertian BRDF.

Features

PBR Shader

• Implemented Cook-Torrance BRDF and Lambertian BRDF.
• Computed diffuse irradiance and glossy irradiance.
• Applied normal mapping and displacement mapping.

Implementation

• GLSL code for PBR shader.
• Implemented diffuse irradiance and glossy irradiance precomputation.
• Applied normal mapping and displacement mapping.
