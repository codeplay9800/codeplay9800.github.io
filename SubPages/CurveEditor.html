Curve Editor

Project Info

C++ OpenGL

Overview

This project is an interactive curve editor and Unity curve plugin developed using the Animation Toolkit.

Features

Spline Curve

• Create and edit different types of spline curves using the GUI.
• Interpolation methods include Linear, Bernstein, Castlejau, Matrix, Hermite, and B-Splines.
• Toggle display of control points and animate a point along the curve.

Rotation

• Animate the orientation of models using Euler angle splines and quaternion splines.
• Interpolation methods include linear and cubic interpolation for Euler angle splines and spherical linear (Slerp) and spherical cubic (Scubic) interpolation for quaternion splines.

Implementation

• Implemented spline curves that can interpolate 3D vector key points.
• Computed control points based on the spline curve's keys and implemented curve interpolation functions.
• Used keyframes to specify changes in the state of an object over time.
