Minecraft Project

Project Info

OpenGL, C++

Overview

This project is a team effort to create a Minecraft-like game with various features such as terrain rendering, procedural generation, and player physics.

Features by: Shubham Sharma

Efficient Terrain Rendering and Chunking

• Interleaved VBOs for faster rendering
• Larger buffers for multiple objects
• Optimized algorithm for creation of terrain mesh

L-Systems

• Modular L-System generator
• Parsed string based on input parameters like axiom, rules, and probability
• Rule class for determining axiom expansion

Shadow Mapping

• Real-time shadow mapping using depth shader
• Addressed issues like shadow acne and Peter Panning
• Poisson sampling with 64-element kernel for anti-aliasing

Water Waves

• Gerstner waves-inspired vertex shader
• Blinn-Phong reflection model for calculating water reflections

Post-process Camera Overlay

• Post-processing water shader using Gaussian blur and FBM noise
• Lava processing shader using FBM noise and vignette effects

Other Features

• Procedural Terrain Generation (Viktoria Braun): Grassy hills biome using Worley noise and fractal Brownian motion, and mountains biome using Perlin noise.
• Player Physics (Manni Arora): Player movements, jumping, and collision detection using grid marching and ray casting.
• Multithreaded Terrain Generation (Manni Arora): Multithreaded terrain generation, using mutexes to ensure parallel threads don't access shared memory simultaneously.
• Texturing and Texture Animation (Viktoria Braun): Texture support, chunk helper, and texture animations for blocks like lava and water.
• Day and Night Cycle (Viktoria Braun): Procedural sky, sun movement, and terrain shader adjustments for a day-night cycle.
• Additional Biomes (Viktoria Braun): Mesa and riverlands biomes, using Perlin noise and smoothstep functions.
• Color Image as Height Map (Manni Arora): Using color images as height maps, incorporating multithreading for quick terrain rendering.
• Distance Fog (Manni Arora): Distance fog, interpolating between fog color and original terrain color based on vertex distance.
• Sounds (Manni Arora): Background bird sounds, block addition and removal sounds, and splash sounds for player interactions.
• NPCs and AI (Manni Arora): NPC spawning, target creation, and pathfinding using grid marching.
