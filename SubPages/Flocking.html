Boid Flocking Simulation

Project Info

C++, OpenGL

Overview

This project is a simulation of Boid Flocking behavior, where boids follow three main rules: cohesion, separation, and alignment.

Features

• Boids try to fly towards the center of mass of neighboring boids (cohesion).
• Boids try to keep a small distance away from other objects (separation).
• Boids try to match velocity with near boids (alignment).

Implementations

Naive Implementation

• Each boid checks all other boids to determine how to modify its velocity.

Uniform Grids

• Dividing the simulation space into cubic cells to reduce the number of boids that need to be checked.

Grids with Memory Coherency

• Optimizing the uniform grid approach by storing the index of each boid's position and velocity in the cell array.
