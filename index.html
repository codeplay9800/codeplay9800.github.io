XR Construct

Project descriptionXR-Construct is built to get all the design team members in the construction process on the same page. We demonstrate BIM(Building Information Modeling) of a architectural model in augmented reality , allowing all (owners, architects, engineers, contractors,subcontractors, and suppliers) to collaborate more accurately and efficiently than using traditional processes. Each Element can be viewed individually and includes other functionalities such as time lapse, Chat system, Information panel The BIM in Augmented Reality can also be used to attract potential buyers by showing them, whole outline of to be constructed (building).

Phantom Limb VR

Interactive and Low-cost method to treat phantom limb pain using Virtual Reality. The project is implemented using google-cardboard, an android device and multiple image markers for 6dof tracking of the VR headset. The virtual experience contains an arm that mimics real body movement using inverse kinematics principles.
