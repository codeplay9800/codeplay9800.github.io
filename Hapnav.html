HapNav

Undergraduate Major Project (8th Semester)

The system uses a combination of 3D space and GPS data to navigate a person inside a building or small expanse with more precession. The user interacts with the environment using a glove which provides feedback in form of force and vibrations. The user receives the directional data in the form of independent feedback over his fingers.
